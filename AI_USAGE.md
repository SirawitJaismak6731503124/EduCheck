AI Tools Used: Gemini (Google AI) & GitHub Copilot

1. What AI Helped Generate
System Architecture (Gemini): Used Gemini to interpret the assignment prompt, define the project structure, and create the technical plan for integrating Flutter with Firebase.

Deployment & Debugging (Gemini): Used Gemini to troubleshoot Firebase Hosting errors (specifically the "Not in a Firebase app directory" error) and for step-by-step CLI commands.

Documentation: Used Gemini to draft the initial Product Requirement Document (PRD) and README content.

2. What I (The Developer) Modified or Implemented
Firebase Configuration: I manually set up the project in the Firebase Console and linked it to the Flutter app using the FlutterFire CLI.

Logic Refinement: I refined the QR scanning logic to ensure the "Check-in" and "Finish Class" steps recorded data to the correct local storage keys.

UI/UX Design: While Copilot suggested the code, I manually adjusted the layout, color scheme, and the "Mood Scale" slider to ensure a user-friendly experience for students.

Deployment Management: I managed the final deployment to Firebase Hosting and handled the Git version control (branching, committing, and pushing).
